# EdenItem 插件说明

## 插件信息
- 插件名：EdenItem
- 作者：非小酋
- 版本：1.0.0
- 简介：瞎写的，不保证能用
- TShock版本：6.0.0
- .NET版本：9

## 指令说明

### /si <物品名称/ID> <玩家/索引/all>
- 权限：eden.si（默认拥有）
- 说明：查找玩家身上是否有指定物品以及数量情况
- 支持的物品位置：背包、银行、银行2、银行3、银行4、虚空袋、垃圾桶
- 示例：
  - `/si 1 玩家1` - 查找玩家1身上的泥土块数量
  - `/si 铁矿 *all*` - 查找所有玩家身上的铁矿数量

### /ri <物品名称/ID> <玩家/索引/all>
- 权限：eden.ri（管理组及以上拥有）
- 说明：移除玩家身上所有的指定物品
- 支持的物品位置：背包、银行、银行2、银行3、银行4、虚空袋、垃圾桶
- 示例：
  - `/ri 1 玩家1` - 移除玩家1身上的所有泥土块
  - `/ri 铁矿 *all*` - 移除所有玩家身上的所有铁矿

## 通配符说明
- 使用 `*all*` 或 `all` 可以选择所有在线玩家
- 例如：`/si 生命水晶 all` - 查找所有玩家身上的生命水晶

## 权限设置
- `eden.si`：允许使用/si指令，默认所有玩家拥有
- `eden.ri`：允许使用/ri指令，默认管理组及以上拥有

## 安装方法
1. 将编译好的EdenItem.dll文件放入TShock的Plugins文件夹
2. 重启TShock服务器
3. 插件会自动加载

## 注意事项
- 使用/ri指令时请谨慎，移除的物品无法恢复
- 插件可能存在bug，使用前请做好备份
- 本插件仅供学习和娱乐使用，请勿用于商业用途
